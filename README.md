## Introduction
> create-panda-app脚手架的内容模板。  
> 模板内容基于vue-cli3.x构建，在此基础上进行一些能力的补充，以便能更好的进行项目的开发和升级。

* [x] axios的封装与代理转发
* [x] vuex的基础能力
* [x] 路由守卫、权限验证
* [x] svg图标组件
* [x] 函数工具库panda-utils
* [x] filters过滤器
